# Amazon UI Clone

A frontend-only Amazon homepage clone built with pure HTML5 and CSS3.

This project focuses on recreating a familiar production-style e-commerce interface while keeping the codebase simple, structured, and beginner-friendly. The UI code is intentionally framework-free and written by hand to demonstrate layout control, component structure, and styling discipline.

## Live Demo

- GitHub Pages: [https://agarwalujala3-lang.github.io/Amazon-UI-Clone/](https://agarwalujala3-lang.github.io/Amazon-UI-Clone/)
- Project preview recording: [Watch demo](./images/Screen%20Recording%202026-02-04%20163818.mp4)

## What This Project Includes

- Amazon-style top navigation with location, account, orders, and cart areas
- Search bar with category dropdown styling and icon-based controls
- Hero banner section with promotional layout and navigation arrows
- Multi-row product card sections inspired by Amazon homepage modules
- Category and image grid presentation using reusable layout patterns
- Large footer with multiple link columns and baseline actions
- Basic responsive adjustments for smaller screen widths

## Run Locally

This is a static frontend project, so no package installation is required.

1. Clone the repository:

   ```bash
   git clone https://github.com/agarwalujala3-lang/Amazon-UI-Clone.git
   ```

2. Open the project folder.
3. Launch `index.html` in a browser.

You can also use a local static server if you want a cleaner development workflow.

## Responsiveness

The project is primarily designed as a desktop-first UI clone.

It already includes basic responsive behavior through CSS media queries, but it is not positioned as a fully production-ready mobile build. The main strength of this repo is layout replication and frontend styling practice.

## Key Frontend Concepts Used

- Flexbox layouts
